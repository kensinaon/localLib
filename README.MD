# Local Library (MDN)

A sample Node+Express+MonggoDB+PUG(Jade) local library tutorial from MDN Web docs. Clone or download as zip.

## Installation & Usage

```bash
git clone https://github.com/kensinaon/local-lib.git
npm install
npm run devstart or npm run start
```
